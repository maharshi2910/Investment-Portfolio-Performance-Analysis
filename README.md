# Investment Portfolio Performance Analysis

## Project Description
This project analyzes an investment portfolio using Python, Pandas, and Matplotlib.

## Features
- Data creation using Pandas
- Save and read CSV files
- Data analysis
- Statistical summary
- Profit/Loss calculation
- Return percentage calculation
- Best and worst performing investment
- Data visualization using charts

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab
